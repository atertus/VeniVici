# Veni Vici!

Welcome to Veni Vici! This application allows you to discover random cat images and their attributes using The Cat API. Enjoy exploring different cat breeds and their unique characteristics.

## Features

- **Random Cat Discovery**: Clicking a button creates a new API fetch request and displays at least three attributes from the returned JSON data.
- **Single View**: Only one item/API call is viewable at a time, ensuring a focused experience.
- **Randomized Results**: API calls appear random to the user, adding an element of surprise.
- **Image Display**: At least one image is displayed per API call, showcasing the cat's appearance.
- **Ban List**: Clicking on a displayed value for one attribute adds it to a displayed ban list.
- **Attribute Filtering**: Attributes on the ban list prevent further images/API results with that attribute from being displayed.
- **Multiple Attribute Types**: Multiple types of attributes can be added to the ban list, providing flexible filtering options.
- **Session History**: Users can see a stored history of their previously viewed items from this session, allowing them to revisit their favorite discoveries.

## How to Use

1. **Discover a Cat**: Click the "Get a Random Cat" button to fetch a random cat image and its attributes.
2. **View Attributes**: Explore the displayed attributes such as breed, weight, origin, and lifespan.
3. **Add to Ban List**: Click on any attribute to add it to the ban list, preventing future cats with that attribute from being displayed.
4. **Check History**: View the history sidebar to see all the cats you've discovered during this session.

Enjoy discovering the wonderful world of cats with Veni Vici!
